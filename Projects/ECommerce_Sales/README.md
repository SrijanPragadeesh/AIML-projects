# E-Commerce Sales Forecasting

A simple machine learning project that predicts e-commerce sales using **Linear Regression**. The notebook covers data preprocessing, model training, evaluation, and visualization of actual vs. predicted sales.

## Features
- Data preprocessing
- Train-test split
- Linear Regression model
- Performance evaluation (MAE, MSE, RMSE, R²)
- Actual vs. Predicted sales visualization

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Run the Project
1. Install the required libraries.
2. Place the dataset (`ECommerce_Sales_Forecasting_Dataset.csv`) in the project folder.
3. Open and run `ECommerce_Sales.ipynb`.

## Output
- Trained sales forecasting model
- Evaluation metrics
- Prediction comparison graph
