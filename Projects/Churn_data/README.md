# Customer Churn Prediction

A machine learning project that predicts customer churn using the **Telco Customer Churn** dataset. The project includes data cleaning, handling missing values, exploratory data analysis (EDA), label encoding of categorical features, feature scaling, and training a **Random Forest Classifier** to classify whether a customer is likely to churn.

## Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Categorical feature encoding
* Feature scaling using `StandardScaler`
* Model training with `RandomForestClassifier`
* Model evaluation using accuracy score

## Tech Stack

* Python
* Pandas & NumPy
* Matplotlib & Seaborn
* Scikit-learn

## Workflow

1. Load and inspect the dataset.
2. Clean and preprocess the data.
3. Perform EDA to understand churn distribution.
4. Encode categorical variables and scale features.
5. Split the data into training and testing sets.
6. Train a Random Forest model.
7. Evaluate the model's performance using accuracy.

This project demonstrates a complete end-to-end machine learning pipeline for solving a binary classification problem.
